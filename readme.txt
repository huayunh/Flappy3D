This game is a rebuild of the popular game Flappy Bird. It includes four mode: classic mode, normal mode, duo mode and the reversed mode.
Classical Mode: Inherit from the original idea of Mr. Dong Nguyen.
Normal Mode: The basic 3D construction, where the user ¡°jumps¡± through all kinds of barriers in a 3D environment.
Duo mode: 2 players support each other to pass the barriers.
Reversed Mode: Use "pipes" to capture the birds in 2D environment.
